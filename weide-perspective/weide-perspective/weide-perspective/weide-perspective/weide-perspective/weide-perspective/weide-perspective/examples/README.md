# 实战案例

本目录存放威德·爱情光谱 Skill 的真实使用案例。

## 案例列表

### 案例 1：聊天冷场后如何挽回
**场景**：男生回复"多休息"后女生只回"嗯"，聊天陷入僵局
**威德分析**：用三层谈话结构分析了问题出在"跳过情绪直接给方案"
**策略输出**：情绪三段式回复（了解→认同→幽默引导）
**结果**：成功挽回聊天节奏，女生主动延伸话题

### 案例 2：第一次约会方案设计
**场景**：双方聊了一个月，第一次正式约会
**威德分析**：判断为第一类约会（展示价值与魅力）
**策略输出**：美术馆+下午茶路线，设计1个亮点（共同的旅行经历）
**结果**：约会顺利，女生主动提出下次见面

### 案例 3：朋友圈分析识人
**场景**：新认识的女生，朋友圈只有风景和书摘，无自拍
**威德分析**：内向丰富型，可能属于"缺认可"人性缺失
**策略输出**：从书摘入手建立深层连接，避免肤浅赞美外表
**结果**：女生被"你比大多数人活得认真"打动，关系快速升温
