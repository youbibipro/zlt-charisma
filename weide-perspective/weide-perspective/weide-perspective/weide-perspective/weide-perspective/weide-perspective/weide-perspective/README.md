## 它解决什么问题

事情是这样的——你在追一个女生，或者刚认识一个感兴趣的女生，但你不知道：

- 她对我有意思吗？怎么判断？
- 该怎么聊天才不尬，还能制造暧昧？
- 约会去哪里、说什么、做什么，才能推进关系？
- 为什么我觉得很用心了，她却越来越远？

市面上有很多"话术"和"套路"，但它们有个共同的问题——**只给鱼，不教渔**。你背了一堆开场白，遇到不同类型的人就不知道用哪个；你学了各种技巧，但没有底层框架，用起来生硬得像在背台词。

**威德·爱情光谱**换了条路：先给你一套完整的认知框架（人性七宗罪→识别女人类型→三层谈话结构→三类约会模型→蜕变成长体系），再基于这套框架给出具体话术和策略。**先知道为什么，再知道怎么做。**

## 效果示例

### 示例 1：分析女性聊天风格并给出策略

**输入**：
> 和这个女生聊了三天，她回复挺积极的但从不主动找我，聊天中说到她工作很忙但喜欢瑜伽和旅行。她朋友圈没有自拍，都是风景和书摘。

**输出**：
```
📊 类型识别
- 物质维度：中性（无炫耀消费）
- 性格维度：内向丰富型（朋友圈无自拍，风景+书摘）
- 人性维度：可能属于"缺认可"类型

🎯 策略建议
1. 第一层策略（展示价值）：分享你的旅行/阅读经历，建立话题连接
2. 第二层策略（情绪对话）：用风景照引发她的感性回忆
3. 第三层策略（意识植入）："你朋友圈分享的书摘我看了，你比大多数人活得认真"

💬 话术推荐
> "你朋友圈那张青海的照片，让我想起在茶卡盐湖看日落的感觉——那种安静里藏着力量的感觉，和你聊天给我的感觉很像。"
```

### 示例 2：约会策略设计

**输入**：
> 约到一个女生周末出来，之前聊得不错，第一次正式约会。

**输出**：
```
📋 约会诊断
- 当前阶段：展示价值与魅力类约会（第一类）
- 核心任务：建立吸引，制造好感

🎯 约会方案
- 地点选择：美术馆/手工课（有话题资源，展示学识品味）
- 黄金30秒：见面不要奔跑迎去，平静直视她眼睛
- 峰终定律设计：安排1-2个亮点（如交换歌单/分享一个有趣的故事）

⚠️ 不要做的事
- 不要选火锅/自助
- 不要争论对错
- 不要问"你觉得怎么样"
```

## 它和同类有什么不同

| 维度 | 同类做法 | 本 Skill |
|:-----|:---------|:---------|
| **核心主张** | 大量话术技巧，强调"怎么撩" | **先修内功再练招式**，强调认知框架 |
| **对女性理解** | 套路化/标签化（"这种女生就爱听这个"） | **动态多面识别**（物质/性格/人性三维度） |
| **聊天方法** | 给固定开场白和标准回复 | **三层谈话结构**（信息→情绪→自我认知） |
| **约会策略** | 约会地点清单 + 话术模板 | **三类约会模型** + 峰终定律 + 意识植入 |
| **底层信仰** | "搞定"思维 | **"吸引"而不是"搞定"**，强调真诚与尊重 |
| **失败处理** | 换下一个话术继续试 | **回测反馈回路**——检查点调整策略 |
| **适可而止** | 无明确边界 | **安全边界**明确，不教操控/欺骗/骚扰 |

3. 话术示例
4. 注意事项（不要做什么）

## 安全边界

这个 Skill **不会**做以下事情：

- ❌ 不教授欺骗/操控/骚扰他人的方法
- ❌ 不鼓励死缠烂打或侵犯边界
- ❌ 不提供万能话术（根本没有这种东西）
- ❌ 不做心理咨询（涉及创伤/心理问题会建议专业帮助）
- ❌ 不承认"搞定"思维的合理性
- ❌ 不分析未成年人或已知非自愿的关系
- ✅ 当用户情绪低落或过度执着时，引导回理性视角
- ✅ 当涉及违法/伤害行为时，明确阻止
